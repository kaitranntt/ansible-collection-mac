Overview
--------

The kaitranntt.mac collection provides comprehensive tools for managing macOS systems with a focus on Tailscale VPN integration. This collection includes roles, modules, and plugins designed specifically for macOS environments.

Features
--------

* Tailscale VPN installation and configuration
* macOS system management
* Automated testing with Molecule
* Quality assurance with pre-commit hooks
* Comprehensive CI/CD pipeline
* Security scanning and compliance
